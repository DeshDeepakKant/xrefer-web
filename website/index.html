About XRefer Visualizer

XRefer is an advanced binary analysis platform that helps security researchers understand complex code interactions and patterns in executable files. This HTML visualization interface provides multiple views into the analyzed binary samples.

The current visualizer is displaying analysis of a BlackCat/ALPHV ransomware variant with WordPress communication capabilities. The sample has been deobfuscated and key functions have been identified.

Use the different visualizations below to explore various aspects of the binary's behavior, structure, and suspicious indicators.

Usage Instructions

Each visualization view provides different interactions:

  • Function Graph: Click on nodes to see details. Use mouse wheel to zoom, drag to pan.
  • Malware Functions: Click on function addresses to jump to that function's details.
  • Memory Map: Hover over regions to see memory access details.
  • Path Graph: Follow execution paths from entry points to suspicious API calls.
  • Complex Analysis: Interactive multi-view with cluster selection and filtering.

Press H in any view to display additional help and keyboard shortcuts.
